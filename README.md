# Eternal Dao PWA

This is the separated-file version of the Eternal Dao game, ready for GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `sw.js`
- `icons/`

## Deploy on GitHub Pages
1. Upload all files and folders to the repository root.
2. In GitHub: **Settings → Pages**.
3. Under **Build and deployment**, set **Source = Deploy from a branch**.
4. Choose **Branch = main** and **Folder = / (root)**.
5. Save.
6. Open the Pages URL on your phone in Chrome.
7. Tap the **INSTALL** button at the top, or use **Add to Home Screen**.

## Notes
- Keep the folder structure unchanged.
- The game saves progress in browser local storage.
- Service worker enables installability and basic offline caching.
